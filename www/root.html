You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /triangle?howMany=num&length=num will generate a number of triangles starting from the length given and slowly shrink based on how many given
  • /circle?howMany=20&radius=20 will generate a number of half circles starting from the radius given and slowly shrink based on how many given
